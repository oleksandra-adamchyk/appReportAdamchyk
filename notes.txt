                    <img src="table.png" alt="Фото коду таблиці" class="screen-img">
                    <img src="table-on-web-site.png" alt="Фото таблиці" class="screen-img">
                    <p>
                        HTML таблиці використовуються для представлення даних у форматі рядків і стовпців, що робить їх
                        зручними для
                        відображення структурованої інформації, такої як таблиці з даними, статистикою чи звітами.
                    </p>
                    <p>
                        <strong>Основні елементи таблиць:</strong>
                    <ul>
                        <li><code>&lt;table&gt;</code>: оголошує саму таблицю.</li>
                        <li><code>&lt;tr&gt;</code> (table row): визначає рядок таблиці.</li>
                        <li><code>&lt;th&gt;</code> (table header): визначає заголовок стовпця (зазвичай текст
                            відображається жирним шрифтом
                            та по центру).</li>
                        <li><code>&lt;td&gt;</code> (table data): визначає клітинку з даними в рядку таблиці.</li>
                    </ul>
                    </p>
                    <p>
                        <strong>Приклад:</strong>
                    </p>
                    <pre>
                            &lt;table&gt;
                                &lt;tr&gt;
                                    &lt;th&gt;Назва&lt;/th&gt;
                                    &lt;th&gt;Ціна&lt;/th&gt;
                                &lt;/tr&gt;
                                &lt;tr&gt;
                                    &lt;td&gt;Продукт 1&lt;/td&gt;
                                    &lt;td&gt;100 грн&lt;/td&gt;
                                &lt;/tr&gt;
                            &lt;/table&gt;
                        </pre>
                </div>
                <div id="list" class="lab1-section content-section text-content">
                    <img src="list.png" alt="Фото коду таблиці" class="screen-img">
                    <img src="list-on-web-site.png" alt="Фото таблиці" class="screen-img">
                    <p>
                        HTML дозволяє створювати списки для структурування контенту. Існує два основних типи списків:
                        нумеровані
                        (упорядковані) і ненумеровані (неупорядковані).
                    </p>
                    <p>
                        <strong>Основні елементи списків:</strong>
                    <ul>
                        <li><code>&lt;ul&gt;</code>: ненумерований список (unordered list).</li>
                        <li><code>&lt;ol&gt;</code>: нумерований список (ordered list).</li>
                        <li><code>&lt;li&gt;</code>: пункт списку (list item).</li>
                    </ul>
                    </p>
                    <p>
                        <strong>Приклад ненумерованого списку:</strong>
                    </p>
                    <pre>
                            &lt;ul&gt;
                                &lt;li&gt;Пункт 1&lt;/li&gt;
                                &lt;li&gt;Пункт 2&lt;/li&gt;
                            &lt;/ul&gt;
                        </pre>
                    <p>
                        <strong>Приклад нумерованого списку:</strong>
                    </p>
                    <pre>
                            &lt;ol&gt;
                                &lt;li&gt;Перше дію&lt;/li&gt;
                                &lt;li&gt;Друге дію&lt;/li&gt;
                            &lt;/ol&gt;
                        </pre>
                </div>
                <div id="images" class="lab1-section content-section text-content">
                    <img src="img.png" alt="Фото коду таблиці" class="screen-img">
                    <img src="img-on-web-site.png" alt="Фото таблиці" class="screen-img">
                    <p>
                        HTML-код зображень використовується для вставлення графіки в веб-сторінку. Зображення можуть
                        бути статичними або
                        анімованими, і вони зазвичай використовуються для ілюстрації контенту, прикраси або надання
                        додаткової інформації.
                    </p>
                    <p>
                        <strong>Основні елементи зображень:</strong>
                    <ul>
                        <li><code>&lt;img&gt;</code>: використовується для вставлення зображення. Основні атрибути:</li>
                        <ul>
                            <li><code>src</code>: вказує URL-адресу зображення.</li>
                            <li><code>alt</code>: альтернативний текст, який відображається, якщо зображення не може
                                бути завантажено, або
                                використовується для доступності.</li>
                        </ul>
                    </ul>
                    </p>
                    <p>
                        <strong>Приклад:</strong>
                    </p>

                    <h3>1. Вбудовані стилі (Inline Styles)</h3>
                    <p>Стилі додаються безпосередньо до HTML-елемента через атрибут <code>style</code>. Це зручно для
                        одноразового
                        форматування елементів, але не рекомендується для великих проєктів.</p>
                    <pre><code>&lt;h1 style="color: blue; font-size: 24px;"&gt;Текст із вбудованим стилем&lt;/h1&gt;</code></pre>
            
                    <h3>2. Внутрішні стилі (Internal Styles)</h3>
                    <p>Внутрішні стилі додаються в документ за допомогою блоку <code>&lt;style&gt;</code> у секції
                        <code>&lt;head&gt;</code>. Це зручно для форматування сторінки, яка не має зовнішнього файлу
                        стилів.
                    </p>
                    <pre><code>&lt;head&gt;
                                    &lt;style&gt;
                                        h1 {
                                            color: red;
                                            font-size: 24px;
                                        }
                                    &lt;/style&gt;
                                &lt;/head&gt;
                                &lt;body&gt;
                                    &lt;h1&gt;Текст із внутрішнім стилем&lt;/h1&gt;
                                &lt;/body&gt;</code></pre>
            
                    <h3>3. Зовнішні стилі (External Styles)</h3>
                    <p>Зовнішні стилі підключаються через окремий CSS-файл за допомогою тега <code>&lt;link&gt;</code> у
                        секції
                        <code>&lt;head&gt;</code>. Це найпоширеніший спосіб, який дозволяє розділити HTML і CSS-код,
                        покращуючи
                        структуру та зручність підтримки.
                    </p>
                    <pre><code>&lt;head&gt;
                                    &lt;link rel="stylesheet" href="styles.css"&gt;
                                &lt;/head&gt;
                                &lt;body&gt;
                                    &lt;h1&gt;Текст із зовнішнім стилем&lt;/h1&gt;
                                &lt;/body&gt;</code></pre>
            
                    <h3>4. Підключення за допомогою @import</h3>
                    <p>Правило <code>@import</code> дозволяє підключати один або більше зовнішніх CSS-файлів у головному
                        CSS-файлі. Це
                        менш популярний метод, оскільки він має певні обмеження в порівнянні з
                        <code>&lt;link&gt;</code>.
                    </p>
                    <pre><code>/* Вміст файлу main.css */
                                @import url("styles.css");
                                
                                h1 {
                                    color: green;
                                }</code></pre>
                </div>

                    <h2>Інші селектори</h2>
            
                    <p>У цьому прикладі ми застосуємо різні типи CSS-селекторів, такі як сусідні, дочірні селектори,
                        селектор атрибута
                        та універсальний селектор.</p>
            
                    <h3>1. Сусідні селектори</h3>
                    <p>Сусідні селектори дозволяють вибирати елементи, які йдуть безпосередньо після іншого елемента.
                    </p>
                    <pre><code>
                                            .service-item + .service-item {
                                                margin-left: 20px;
                                            }
                                                </code></pre>
                    <p>У цьому прикладі кожен елемент з класом <code>.service-item</code> отримає лівий відступ, якщо
                        перед ним є інший
                        елемент з таким самим класом.</p>
            
                    <h3>2. Дочірні селектори</h3>
                    <p>Дочірні селектори дозволяють вибирати елементи, які є прямими нащадками іншого елемента.</p>
                    <pre><code>
                                            .service-grid > .service-item {
                                                border: 1px solid #ccc;
                                                padding: 10px;
                                            }
                                                </code></pre>
                    <p>У цьому прикладі всі елементи з класом <code>.service-item</code>, які є прямими нащадками
                        елемента з класом
                        <code>.service-grid</code>, отримають обведення та відступи.
                    </p>
            
                    <h3>3. Селектор атрибута</h3>
                    <p>Селектор атрибута дозволяє вибирати елементи за значенням їх атрибута.</p>
                    <pre><code>
                                            input[type="email"] {
                                                border-color: #3498db;
                                            }
                                                </code></pre>
                    <p>У цьому прикладі ми використовуємо селектор атрибута для вибору всіх елементів
                        <code>&lt;input&gt;</code> з
                        атрибутом <code>type="email"</code> і встановлюємо для них синій колір рамки.
                    </p>
            
                    <h3>4. Універсальний селектор</h3>
                    <p>Універсальний селектор вибирає всі елементи на сторінці.</p>
                    <pre><code>
                                            * {
                                                box-sizing: border-box;
                                                margin: 0;
                                                padding: 0;
                                            }
                                                </code></pre>
                    <p>Цей універсальний селектор скидає відступи і додає <code>box-sizing: border-box;</code> до всіх
                        елементів на
                        сторінці, що допомагає в керуванні розмірами елементів.</p>

                                                .content-section {

                                                h1 {

                    <h2>Висновки</h2>
                    <p>
                        У процесі виконання роботи було отримано практичні навички використання каскадних таблиць стилів
                        (CSS) для
                        оформлення HTML-документів. Ми розглянули та реалізували такі методи підключення стилів:
                    </p>
                    <ul>
                        <li>Вбудовані стилі (inline styles) – для одноразового форматування елементів.</li>
                        <li>Внутрішні стилі (internal styles) – для роботи з окремими сторінками.</li>
                        <li>Зовнішні стилі (external styles) – для організації та підтримки великих проєктів.</li>
                        <li>Метод <code>@import</code> – для підключення додаткових файлів стилів.</li>
                    </ul>
                    <p>
                        Вивчено принципи використання селекторів: тегів, класів, ідентифікаторів, атрибутів, дочірніх та
                        сусідніх
                        елементів.
                    </p>
                    <p>
                        Також застосовано різноманітні властивості CSS для стилізації текстових елементів, списків, фону
                        та
                        інтерактивних ефектів.
                        Висновок: використання CSS дозволяє створювати привабливі, функціональні та зручні для
                        користувачів
                        веб-сторінки.
                    </p>
                    <h2>Контрольні питання</h2>
            
                    <h3>1. Для чого використовуються каскадні таблиці стилів?</h3>
                    <p>Каскадні таблиці стилів (CSS) використовуються для опису зовнішнього вигляду елементів
                        веб-сторінки, таких як шрифти,
                        кольори, відступи, розміри тощо. Вони дозволяють розділити структуру документа (HTML) і його
                        стилі (CSS),
                        забезпечуючи гнучкість і зручність у дизайні.</p>
            
                    <h3>2. Які методи визначення стилів у HTML-документі ви знаєте?</h3>
                    <ul>
                        <li>Вбудований стиль (inline styles): за допомогою атрибуту <code>style</code> у тегах HTML.
                        </li>
                        <li>Внутрішній стиль (internal styles): у межах тегів <code>&lt;style&gt;</code> у розділі
                            <code>&lt;head&gt;</code>.
                        </li>
                        <li>Зовнішній стиль (external styles): у зовнішньому CSS-файлі, підключеному через тег
                            <code>&lt;link&gt;</code>.
                        </li>
                    </ul>
            
                    <h3>3. У чому різниця між принципами каскадування і спадкування?</h3>
                    <p>
                        <strong>Каскадування:</strong> Це процес вирішення конфліктів між стилями, що застосовуються до
                        одного і того ж
                        елемента з різних джерел. Враховується специфічність, порядок визначення і важливість
                        (important).<br>
                        <strong>Спадкування:</strong> Це властивість деяких CSS-властивостей (наприклад, колір тексту
                        або шрифт) автоматично
                        передаватися дочірнім елементам.
                    </p>
            
                    <h3>4. Як в HTML-документі оформлюються селектори тегу і класу?</h3>
                    <p>
                        <strong>Селектор тегу:</strong> Оформлюється через назву тега, наприклад:
                    <pre><code>
                                        p {
                                            color: blue;
                                        }
                                        </code></pre>
                    <strong>Селектор класу:</strong> Починається з крапки і застосовується до елементів з відповідним
                    атрибутом
                    <code>class</code>:
                    <pre><code>
                                        .highlight {
                                            background-color: yellow;
                                        }
                                        </code></pre>
                    </p>
            
                    <h3>5. Для чого використовується ідентифікатор?</h3>
                    <p>Ідентифікатор використовується для стилізації або маніпуляцій із певним унікальним елементом
                        HTML-сторінки. Він
                        задається через атрибут <code>id</code> і викликається в CSS за допомогою символу
                        <code>#</code>.
                    </p>
            
                    <h3>6. Навести приклади оформлення текстових елементів</h3>
                    <p>Приклади стилізації тексту:</p>
                    <pre><code>
                                    h1 {
                                        font-size: 24px;
                                        color: #2c3e50;
                                        text-align: center;
                                    }
                                
                                    p {
                                        line-height: 1.6;
                                        font-family: 'Arial', sans-serif;
                                        color: #7f8c8d;
                                    }
                                
                                    .highlight {
                                        font-weight: bold;
                                        color: #e74c3c;
                                        background-color: #fef9e7;
                                    }
                                    </code></pre>

                    <h2>Опис особливостей макету</h2>
            
                    <h3>1. Блокова гумова верстка</h3>
                    <p>
                        Макет побудований за принципом блокової верстки, де основні елементи, такі як
                        <code>header</code>,
                        <code>section</code>,
                        <code>footer</code>, розташовані у вигляді окремих блоків. Кожен блок займає певну частину
                        сторінки, але їх
                        розміри
                        змінюються відповідно до ширини екрану, що є ознакою гумової верстки. Це досягається
                        використанням відносних
                        одиниць,
                        таких як <code>%</code>, <code>vh</code>, <code>vw</code>.
                    </p>
                    <pre><code>
                                                                                /* Приклад гумової верстки */
                                                                                .container {
                                                                                    width: 90%; /* Ширина контейнера змінюється відповідно до ширини вікна */
                                                                                    margin: 0 auto;
                                                                                    max-width: 1200px; /* Максимальна ширина для великих екранів */
                                                                                }
                                                                            </code></pre>
            
                    <h3>2. Використання Flexbox</h3>
                    <p>
                        У макеті активно використовується Flexbox для вирівнювання елементів. Наприклад, у шапці сайту
                        логотип і меню
                        навігації
                        вирівнюються за допомогою властивостей <code>display: flex</code> і
                        <code>justify-content</code>.
                    </p>
                    <pre><code>
                                                                                /* Шапка сайту */
                                                                                header {
                                                                                    display: flex;
                                                                                    justify-content: space-between; /* Логотип зліва, меню справа */
                                                                                    align-items: center; /* Вирівнювання по вертикалі */
                                                                                    padding: 20px;
                                                                                    background-color: #f5f5f5;
                                                                                }
                                                                        
                                                                                /* Секція карток */
                                                                                .card-container {
                                                                                    display: flex;
                                                                                    flex-wrap: wrap; /* Елементи переходять на новий рядок при недостатній ширині */
                                                                                    gap: 20px; /* Відстань між картками */
                                                                                }
                                                                        
                                                                                .card {
                                                                                    flex: 1 1 calc(33.333% - 20px); /* Три картки в ряд */
                                                                                    max-width: 300px; /* Обмеження максимальної ширини */
                                                                                    background-color: #fff;
                                                                                    border: 1px solid #ddd;
                                                                                    padding: 15px;
                                                                                }
                                                                            </code></pre>
            
                    <h3>3. Адаптивна верстка</h3>
                    <p>
                        Хоча макет побудований за принципом гумової верстки, його можна легко адаптувати для різних
                        розмірів екранів за
                        допомогою
                        медіазапитів. Наприклад, у наведеному коді картки автоматично змінюють свій розмір, але можна
                        додати медіазапити
                        для
                        кращої адаптації.
                    </p>
                    <pre><code>
                                                                                @media (max-width: 768px) {
                                                                                    .card {
                                                                                        flex: 1 1 100%; /* Картки займають всю ширину екрана на малих пристроях */
                                                                                    }
                                                                                }
                                                                            </code></pre>

                        відносним одиницям,
                        і адаптивним, оскільки підтримує зміну вигляду елементів залежно від ширини екрана. Flexbox
                        забезпечує зручність

                    // Перевірка правильності закриття дужок у рядку
                    function checkBrackets(str) {
                        const stack = [];
                        const brackets = {
                            "(": ")",
                            "{": "}",
                            "[": "]",
                        };
                    
                        for (const char of str) {
                            if (brackets[char]) {
                                stack.push(char); // Відкриваюча дужка додається в стек
                            } else if (Object.values(brackets).includes(char)) {
                                const last = stack.pop(); // Порівнюємо зі стеком
                                if (brackets[last] !== char) return false; // Помилка, якщо дужки не збігаються
                            }
                        }
                    
                        return stack.length === 0; // Перевіряємо, чи всі дужки закриті
                    }
                    
                    // Приклад використання:
                    console.log(checkBrackets("someFn({})")); // true
                    console.log(checkBrackets("someFn({[)]}")); // false
